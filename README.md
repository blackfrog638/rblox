# rblox

Yet another Rust implementation of the Lox interpreter.

## Features

- Lexer, parser, and AST
- Basic expression evaluation
- REPL / script entry point

## Run

```bash
cargo run
```

To run a script file:

```bash
cargo run -- path/to/script.lox
```

- Rust 2021
- Entry point: `src/main.rs`
